# MGRAG — СтройПроект (ранее ПМООС-RAG)

Локальная (офлайн) RAG-система для автоматизации ответов на замечания
государственной экспертизы по разделу **ПМООС/ООС** проектной документации
(Постановление Правительства РФ № 87). Анализирует проектную документацию,
формирует ответы на замечания, готовит откорректированные тома ООС и выгрузку
для УПРЗА «Эколог» (ИНТЕГРАЛ). Данные хранятся локально; обработка оптимизирована
под ноутбук с GPU (RTX 3070 Ti, 8 ГБ VRAM).

**Текущая версия:** v0.21.0 «Measured».

## Стек

Python 3.11 / Streamlit · эмбеддер BAAI/bge-m3 · реранкер BAAI/bge-reranker-v2-m3 ·
Qdrant (embedded) · гибридный поиск (dense + BM25 + RRF + реранк) · провайдеры
LLM на выбор (DeepSeek, OpenAI, Anthropic, Gemini, Ollama, Mistral, Kimi).

## Установка и запуск (Windows)

```bat
cd V0.21.0
install.bat
run.bat                  :: весь хаб (6 модулей во вкладках)
run_ui_module.bat        :: один модуль как отдельное Streamlit-приложение
```

Linux — `install.sh` / `run.sh`. Отдельные модули в CLI — `run_module.bat`.
Ключи API задаются в `.env` (шаблон — `.env.example`).

## Модули

1. Систематизация ПД (ПП-87) · 2. Индексация (RAG-база) · 3. Граф связей ·
4. Ответы на замечания ООС · 5. Корректировка томов ООС · 6. Выгрузка для УПРЗА.
