# СтройПроект v0.31.0 «Portable»

Дата: 23.07.2026. Накопительный релиз после серии рабочих правок 0.30.1–0.30.6
(по живой эксплуатации на проекте ОПОЧКА-ДУБРОВКА).

## Как обновиться
- `git pull` в папке приложения — install.bat **НЕ нужен** (зависимости не менялись).
- Либо распаковать нумерованный релиз `releases/STR.RAG-v0.31.0.zip`.
- Начиная с этой версии run.bat **сам предупредит**, если после обновления
  install.bat всё же понадобится (сверяет отпечаток зависимостей).

## Главные новшества

### Одна база на несколько компьютеров (заголовочная фича)
- **Модуль 2 → «💾 Одна база на несколько компьютеров»**: кнопки «📤 Выгрузить»
  и «📥 Забрать» переносят всю базу через OneDrive (по умолчанию) или указанную
  папку. Живая база остаётся локальной (в OneDrive напрямую держать нельзя —
  синхронизация на лету портит Qdrant/sqlite).
- Защита: манифест целостности (недокачанная облаком копия не ставится),
  страховочная копия с автооткатом, замок базы на время выгрузки.
- Запасной путь без интерфейса — скрипты `база_в_облако.bat` / `база_из_облака.bat`.

### Отзыв неверных ответов из памяти
- **Модуль 7 → «🧠 Память экспертизы»**: неудачный ответ можно «Отозвать» — он
  перестаёт подмешиваться как пример к будущим ответам. Запись сохраняется
  (аудит-след); тот же неизменённый ответ сам не вернётся; исправленный — снимает
  отзыв. Закрывает риск «принятый в спешке неверный ответ размножается».

### Устойчивость (по живым сбоям)
- Единый кэш моделей + строго-офлайн загрузка: «вечная загрузка модели» и
  повторное скачивание 2.3 ГБ через прокси устранены.
- Краш-маркер индексации (недописанный при сбое файл переиндексируется целиком).
- Префлайт диска (<1 ГБ свободного — понятный отказ вместо падения).
- Тип объекта (площадной/линейный) фиксируется за проектом при первом запуске —
  «Продолжить» больше не смешивает две разметки разделов ПП-87.

### Процесс релизов (как в ЭКО.DOC)
- `собрать_релиз.bat` / `scripts/build_release.py` собирают
  `releases/STR.RAG-vX.Y.Z.zip` — имя файла всегда совпадает с версией в коде.

Полный перечень правок — в истории коммитов (0.30.1 … 0.31.0).
