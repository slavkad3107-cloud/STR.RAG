# ПМООС-RAG v0.18.0 «Split» — резюме изменений

**Дата:** 27.06.2026
**Базовая версия:** v0.17.0 «Faster» (все её оптимизации включены).
**Состав поставки:** `V0.18.0/` (исходники) + `MGRAG V0.18.0.zip`
(внутренняя папка `V0.18.0`) + это РЕЗЮМЕ.

Релиз делает **модули запускаемыми по отдельности**. Подход **аддитивный**:
единый интерфейс (`run.bat` → весь хаб) не изменён по поведению; добавлены
отдельные приложения модулей, переиспользующие те же функции.

---

## Что нового (модульный запуск)

1. **`app/hub.py` сделан импортируемым** без побочных эффектов: `st.set_page_config`
   вынесен с уровня модуля в `main()`, CSS размера шрифта — в функцию
   `apply_font_css(cfg)`. Поведение единого хаба не изменилось.
2. **Отдельные Streamlit-приложения модулей** — `app/modules_ui/module1..6.py`.
   Каждое переиспользует `sidebar()` и `tab_mN()` из хаба (без дублирования
   логики). Данные/проекты/индекс — общие (каталог `%USERPROFILE%\.pmoos-rag`).
3. **Лаунчер `run_ui_module.bat`** — меню выбора модуля → `streamlit run
   app\modules_ui\moduleN.py` (с активацией общего venv, как в `run.bat`).

Папка `app/modules_ui` намеренно НЕ называется `pages/`, чтобы Streamlit не
подхватывал её как multipage и не менял вид единого хаба.

## Как запускать

- **Весь хаб (как раньше):** `run.bat` → `streamlit run app/hub.py`.
- **Один модуль (Streamlit):** `run_ui_module.bat` (меню) или напрямую
  `streamlit run app/modules_ui/module4.py`.
- **Один модуль (CLI, как раньше):** `run_module.bat` → `modules/*.py`.

## Включает (из v0.17.0 «Faster»)

FP16 реранкера (эмбеддер opt-in), синглтон моделей на процесс, параллельное
расширение запросов в М4 + прогрев кэша эмбеддингов, кэш базы памяти, BM25 с
нормативными кодами, JSON-ремонт с повтором. Подробнее — РЕЗЮМЕ_v0.17.0.md.

## Проверка

Выполнено: `py_compile` всех файлов; импорт `app.hub` без побочных эффектов;
**headless Streamlit AppTest** хаба и всех 6 модульных приложений — загрузка и
полный рендер с проектом без исключений и без дублей ключей виджетов;
адверсариальный мультиагентный ревью (правка лаунчера по итогам: `pause` после
остановки модуля).

**На вашем ПК (RTX 3070 Ti) перед боевым использованием:** живой прогон
`run.bat` и `run_ui_module.bat` (выбрать модуль, открыть проект, прогнать
сценарий М1→М6) — проверить интерактив с реальными данными/моделями.

## Откат
v0.18.0 аддитивна: чтобы вернуться к единому хабу без модульных приложений —
удалить `app/modules_ui/` и `run_ui_module.bat`; `app/hub.py` работает и так
(импортируемость безвредна для единого запуска).
