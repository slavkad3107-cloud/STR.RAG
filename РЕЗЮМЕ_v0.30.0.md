# СтройПроект v0.30.0 «Context» — резюме изменений

**Дата:** 07.07.2026 · **База:** v0.29.0 «Complete» · **Архив:** `STR.RAG V0.30.0.zip`

Три содержательные фичи из роадмапа качества (последние крупные из очереди
ревью). Поведение по умолчанию не меняется — рискованное включается флагом
после A/B.

## 1. Связные таблицы при нарезке (по умолчанию ВКЛ, для новой индексации)

Длинная таблица (выбросы, отходы по ФККО) разрезалась на несколько чанков, и
куски-продолжения теряли строку заголовка — поиск находил «0.023 | 1.7» без
смысла колонок, связка «код → вещество → значение» рвалась (RAG уверенно «врал»
по числам). Теперь к каждому чанку-продолжению приклеивается заголовок таблицы
с пометкой «(продолжение таблицы)». Вступает в силу при следующей индексации/
переиндексации; на уже построенные базы не влияет.

## 2. Contextual retrieval (opt-in: `chunking.contextual`)

«Немой» обрывок таблицы без слов «ПМООС/раздел 8» невозможно найти ни
семантикой, ни лексикой. При включённом флаге каждый чанк получает префикс
`[ПМООС · Перечень мероприятий… · файл.pdf]`, который участвует в ЭМБЕДДИНГЕ и
BM25 — но показ-текст (для ИИ и интерфейса) остаётся чистым. По умолчанию ВЫКЛ:
включать после A/B на golden-set (М7 → команда замера → М2 reindex с флагом →
замер → сравнить). ID чанков от флага не зависят.

## 3. Слой статусов НПА (реестр 12 → 27 записей)

Главная содержательная претензия эксперта — ссылка на отменённый документ.
Реестр дополнен базовыми НПА ООС со статусами (действует/заменён/отменён):
ФЗ-7/89/96, СанПиН 2.1.3684-21, перечень ЗВ 2909-р, ГОСТ 17.2.3.02-78→2014,
СНиП 11-02-96→СП 47, СНиП 23-03-2003→СП 51, НМУ (РД 52.04.52-85→Приказ МПР
№811) и отменённые «гильотиной» 2021 года: ГН 2.1.6.3492-17 и ГН 2.2.5.3532-18
(→ СанПиН 1.2.3685-21), СанПиН 2.1.4.1074-01, СП 2.1.5.1059-01. Исправлена
само-ссылка СП 51.13330.2011 (ложно помечался «заменённым»). Проверка ссылок
уже встроена в М4 (consistency) и Блок 2 (рецензия) — теперь она знает больше.
Реестр расширяется пользователем без кода (data/normatives.yaml + свой YAML).
